# 🛒 Ecommerce Website

An interactive ecommerce website built using HTML, CSS, and JavaScript. It includes basic shopping functionality like quantity selection and "Add to Cart" logic.

## 🔍 Features

- Multi-page layout (Home, Product)
- Add to Cart functionality (with quantity input)
- Basic product showcase
- Clean UI using HTML & CSS
- JavaScript used for cart actions (alert-based)

## 📂 Pages Included

- `index.html` - Home page
- `product.html` - Product detail page (with Add to Cart)
- `cart.html` - (To be enhanced in future)
- `style.css` - Styling
- `script.js` - JavaScript logic
